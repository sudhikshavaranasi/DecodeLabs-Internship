# Project 3 - Career Recommendation System

## Objective
Develop a simple AI-based recommendation system that suggests suitable career paths based on a user's skills and interests.

## Description
This project uses a content-based recommendation approach. The system compares user-entered skills with predefined skill requirements for various careers and recommends the most suitable career options based on matching scores.

## Features
- Accepts user skills/interests as input
- Matches skills with career requirements
- Calculates similarity scores
- Displays recommended careers in ranked order
- Simple and user-friendly interface

## Technologies Used
- Python

## How It Works
1. User enters skills or interests.
2. The system compares them with stored career profiles.
3. A match score is calculated.
4. Careers are ranked based on the score.
5. The best matches are displayed.

## Example Input
coding, problem solving, algorithms

## Example Output
Recommended Careers:

Software Engineer (Match Score: 3)

## Run the Program

```bash
python career_recommendation.py
```
